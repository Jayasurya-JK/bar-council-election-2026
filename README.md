# Bar Council Election 2026 - Modern Web App

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000)

## Vercel Deployment

1. Push to GitHub
2. Import project in Vercel
3. Deploy (zero configuration needed)

## Features

- ⚡ **Instant Search** with debounced autocomplete
- 📱 **Fully Responsive** mobile-first design
- 🎨 **Modern UI** with Tailwind CSS gradients and animations
- 🚀 **Serverless API** for efficient CSV searching
- ✨ **Premium Design** with glow effects and smooth transitions

## Tech Stack

- Next.js 15 (App Router)
- React 19
- TypeScript
- Tailwind CSS
- PaPaParse (CSV parsing)
